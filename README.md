# OpenStack-Orchestration-Infrastructure-as-Code

Dans le cadre d'un projet académique en binôme, j'ai orchestré le déploiement automatisé d'une infrastructure Cloud OpenStack complète.
Déploiement automatisé : Utilisation de Kolla-Ansible pour conteneuriser et déployer les services OpenStack (Nova, Glance, Neutron, Manila). Configuration multinode avec Docker, MariaDB et HAProxy.
Évolution dynamique : Ajout de compute nodes et extension des services via playbooks Ansible. Gestion complète du cycle : inventaire, déploiement ciblé et validation.
Stacks réutilisables : Création de templates IaC pour instances avec volumes persistants, IP flottantes, architecture bastion sécurisée, et déploiement scalable multi-instances.
